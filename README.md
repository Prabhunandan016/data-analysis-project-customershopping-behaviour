# Consumer Shopping Behavior Analysis

## Project Overview

A leading retail company aims to better understand customer shopping behavior to improve **sales, customer satisfaction, and long-term loyalty**. This project analyzes consumer shopping data to uncover patterns in purchasing behavior across **demographics, product categories, and sales channels (online vs offline)**.

The goal is to answer the business question:

**"How can the company leverage consumer shopping data to identify trends, improve customer engagement, and optimize marketing and product strategies?"**

This project uses **Python, SQL, and Power BI** to clean, analyze, and visualize consumer behavior data to generate actionable business insights.

---

# Project Objectives

* Analyze consumer purchase behavior across different demographics.
* Identify key drivers influencing purchasing decisions such as **discounts, seasons, reviews, and payment methods**.
* Understand **customer loyalty and repeat purchasing patterns**.
* Compare **online vs offline shopping trends**.
* Provide **data-driven recommendations** to improve marketing strategies and customer engagement.

---

# Project Workflow

## 1. Data Preparation & Modeling (Python)

Python was used to clean, preprocess, and prepare the dataset for analysis.

Tasks performed:

* Data cleaning and handling missing values
* Data type conversions
* Feature engineering
* Preparing structured datasets for SQL analysis

Tools used:

* Pandas
* NumPy
* Jupyter Notebook / Python scripts

---

## 2. Data Analysis (SQL)

The dataset was organized into structured tables and SQL queries were used to extract insights.

Key analyses performed:

* Customer segmentation
* Purchase frequency analysis
* Impact of discounts on sales
* Average product review analysis
* Seasonal purchase trends
* Payment method preferences

Technologies used:

* PostgreSQL
* pgAdmin4

---

## 3. Data Visualization (Power BI)

An interactive Power BI dashboard was developed to visualize key patterns in customer behavior.

Dashboard features:

* Sales performance by product category
* Customer demographics analysis
* Online vs offline purchase comparison
* Seasonal purchasing trends
* Top rated products and categories

The dashboard helps stakeholders quickly identify **trends and business opportunities**.

---

## 4. Insights & Business Recommendations

Key insights generated from the analysis include:

* **Discounts significantly increase purchase probability**
* **Highly reviewed products show stronger sales performance**
* **Seasonal trends influence product demand**
* **Online purchases are growing faster compared to offline channels**
* **Certain customer segments demonstrate higher repeat purchase behavior**

Recommended strategies:

* Personalized promotions for high-value customers
* Improve product visibility for highly rated items
* Seasonal marketing campaigns
* Loyalty programs to increase repeat purchases
